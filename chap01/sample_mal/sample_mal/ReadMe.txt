========================================================================
    Win32 应用程序: sample_mal 工程概况
========================================================================

sample_mal 应用程序是通过 AppWizard 生成的。

这份文档中包含 sample_mal 应用程序中各文件的内容概况。


sample_mal.vcxproj
    这是由 AppWizard 生成的 VC++工程的主工程文件。
    其中包含生成该文件的 Visual C++ 版本，AppWizard中选择的平台/组成，
    以及工程功能相关的信息。

sample_mal.vcxproj.filters
    这是由 AppWizard 生成的 VC++ 工程的过滤器文件。 
    其中包含工程中各文件和过滤器之间的关系信息，用于在IDE中将文件按照指定顺序和扩展名分组。
    例如，".cpp"文件就关联了"源文件"过滤器。

sample_mal.cpp
    这是应用程序主要的源文件。

/////////////////////////////////////////////////////////////////////////////
AppWizard 还生成了以下资源文件。

sample_mal.rc
    这是程序所使用的所有 Microsoft Windows 资源一览。
    其中引用了RES子目录下的图标、位图、光标等文件。
    这个文件可以用 Microsoft Visual C++ 直接编辑。

Resource.h
    这是定义新资源 ID 的标准头文件。
    这个文件的载入和更新由 Microsoft Visual C++ 来完成。

sample_mal.ico
    这是应用程序的图标 (32x32) 文件。
    主要源文件 sample_mal.rc 引用了这个文件。

small.ico
    这是应用程序的小图标 (16x16) 文件。
    主要源文件 sample_mal.rc 引用了这个文件。

/////////////////////////////////////////////////////////////////////////////
其他标准文件 :

StdAfx.h、StdAfx.cpp
    这些文件用于生成预编译头文件 sample_mal.pch 和预编译类型文件 StdAfx.obj。

/////////////////////////////////////////////////////////////////////////////
其他说明 :

AppWizard 会使用 "TODO:" 注释来标记用户需要添加或自定义的源代码。
