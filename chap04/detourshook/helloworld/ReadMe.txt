========================================================================
    控制台应用程序: helloworld 工程概况
========================================================================

helloworld 应用程序是通过 AppWizard 生成的。

这份文档中包含 helloworld 应用程序中各文件的内容概况。


helloworld.vcxproj
    这是由 AppWizard 生成的 VC++工程的主工程文件。
    其中包含生成该文件的 Visual C++ 版本，AppWizard中选择的平台/组成，
    以及工程功能相关的信息。

helloworld.vcxproj.filters
    这是由 AppWizard 生成的 VC++ 工程的过滤器文件。 
    其中包含工程中各文件和过滤器之间的关系信息，用于在IDE中将文件按照指定顺序和扩展名分组。
    例如，".cpp"文件就关联了"源文件"过滤器。

helloworld.cpp
    这是应用程序主要的源文件。

/////////////////////////////////////////////////////////////////////////////
其他标准文件 :

StdAfx.h、StdAfx.cpp
    这些文件用于生成预编译头文件 helloworld.pch 和预编译类型文件 StdAfx.obj。
